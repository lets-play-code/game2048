How to use

先在表格中修改当前库存，再点击 Next Day。页面不会保留逐日历史，只展示这一刻的库存状态。默认样例与测试侧 TestFixture 一致，因此你可以在文本入口和 Web 入口之间来回对照。页面也会如实呈现当前遗留实现的行为，包括 Conjured Mana Cake 还没有被特殊处理这一点。
