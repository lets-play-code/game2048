# 保存 2048 排行榜成绩

## 被测接口

- `POST /api/games/{id}/leaderboard`

## 输入因子

| 因子 | 取值 |
| --- | --- |
| 游戏是否已存在 | 不存在；已存在 |
| 游戏状态 | 未终局；失败终局；胜利终局；已记录过成绩 |
| `playerName` | 缺失；仅空白字符；普通姓名 |
| 排行榜现有记录 | 无；同名且旧分更低；同名且旧分更高 |
| 外部 wall API | `200`；`500` |

## 输出因子

| 因子 | 取值 |
| --- | --- |
| HTTP 状态 | `400`、`200`、`500` |
| 错误信息 | 未终局不可保存；`playerName` 缺失；当前局已记录过成绩 |
| 排行榜表 | 不写入；新增；更新为更高分；保留旧最佳成绩 |
| 当前游戏 `recordSaved` | 外部调用成功后置为 `true`；外部调用失败时保持 `false` |

## 用例

| 用例名 | 关键输入 | 期望输出 |
| --- | --- | --- |
| 懒创建但未终局 | `id` 不存在，默认新局 | 返回 `400`，排行榜不写入 |
| 缺失玩家名 | 终局失败局，请求体无 `playerName` | 返回 `400`，排行榜不写入 |
| 空白名保存成功 | 终局失败局，`playerName="   "`，wall 成功 | 返回 `200`，保存空白名并将当前局标记为已记录 |
| 更高分覆盖同名记录 | 同名旧分较低，当前胜利分数更高，wall 成功 | 返回 `200`，最佳成绩更新为更高分 |
| 更低分不覆盖同名记录 | 同名旧分较高，当前失败分数更低，wall 成功 | 返回 `200`，排行榜保留旧最佳成绩 |
| 已记录过成绩 | 当前局 `scoreRecorded=true` | 返回 `400`，排行榜不变 |
| wall 调用失败 | 终局失败局，wall 返回 `500` | 接口返回 `500`，但排行榜先写入，当前局仍未标记为已记录 |
