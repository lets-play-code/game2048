# 创建 2048 游戏

## 被测接口

- `POST /api/games`

## 输入因子

| 因子 | 取值 |
| --- | --- |
| 请求体 | `{}` |
| 现有游戏 | 不存在 |

## 输出因子

| 因子 | 取值 |
| --- | --- |
| HTTP 状态 | `200` |
| `body.json.id` | 32 位十六进制字符串 |
| `body.json.state` | 新局默认状态：`score=0`、`win=false`、`lose=false`、`gameOver=false` |
| `body.json.state.tiles.value[]` | 4x4 棋盘，14 个空格与 2 个初始 `"2"` |

## 用例

| 用例名 | 关键输入 | 期望输出 |
| --- | --- | --- |
| 默认创建新局 | 空请求 | 返回新 id、默认分数文案、默认棋盘布局 |
