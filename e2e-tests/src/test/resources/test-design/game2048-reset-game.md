# 重置 2048 游戏

## 被测接口

- `POST /api/games/{id}/reset`

## 输入因子

| 因子 | 取值 |
| --- | --- |
| 游戏是否已存在 | 不存在；已存在 |
| 已有游戏状态 | 无；胜利且已记录成绩的终局 |

## 输出因子

| 因子 | 取值 |
| --- | --- |
| HTTP 状态 | `200` |
| 返回游戏状态 | 默认新局 |
| 存档表 | 始终为空，不会因为 reset 写入或刷新 |

## 用例

| 用例名 | 关键输入 | 期望输出 |
| --- | --- | --- |
| 懒创建并重置 | `id` 不存在 | 返回默认新局且不写入存档 |
| 清空旧终局状态 | 已存在胜利且已记录成绩的局面 | 返回默认新局，终局标记清空，存档仍不变 |
