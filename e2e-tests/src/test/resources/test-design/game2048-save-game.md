# 保存 2048 游戏

## 被测接口

- `POST /api/games/{id}/save/{slotKey}`

## 输入因子

| 因子 | 取值 |
| --- | --- |
| `slotKey` | 非法槽位 `slot9`；合法槽位 `auto`；合法手动槽位 `slot2` |
| 游戏是否已存在 | 不存在；已存在 |
| 已有游戏状态 | 默认新局；胜利且已记录成绩；普通进行中局面 |
| 目标槽位是否已有记录 | 无记录；已有记录 |

## 输出因子

| 因子 | 取值 |
| --- | --- |
| HTTP 状态 | `400` 或 `200` |
| 返回游戏状态 | 默认新局；原有进行中局面；已记录成绩的胜利局 |
| 存档表 | 不写入；新建记录；覆盖旧记录 |

## 用例

| 用例名 | 关键输入 | 期望输出 |
| --- | --- | --- |
| 非法槽位 | `slotKey=slot9` | 返回 `400`，存档表保持空 |
| 懒创建后保存 auto | `id` 不存在，`slotKey=auto` | 返回默认新局，并生成 auto 存档 |
| 保存已记录胜利局到手动槽位 | 胜利且 `scoreRecorded=true`，`slotKey=slot2` | 返回终局状态，并保存完整棋盘与终局标记 |
| 覆盖已有手动槽位 | 目标 `slot2` 已有旧记录 | 返回当前局面，并用当前局面覆盖旧记录 |
